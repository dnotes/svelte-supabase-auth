Supabase Auth Web Component

This page demonstrates how to use the Supabase Auth component as a web component in any HTML page.

Basic Usage

Include the web component script and use the custom element:

<script type="module" src="https://unpkg.com/svelte-supabase-auth@latest/web-component"></script>

<supabase-auth
  supabase-url="https://your-project.supabase.co"
  supabase-anon-key="your-anon-key"
></supabase-auth>

With Social Providers

Configure social authentication providers:

<supabase-auth
  supabase-url="https://your-project.supabase.co"
  supabase-anon-key="your-anon-key"
  providers='["google", "github", "discord"]'
  social-layout="horizontal"
  social-button-size="large"
></supabase-auth>

With Custom Configuration

Customize authentication options and internationalization:

<supabase-auth
  supabase-url="https://your-project.supabase.co"
  supabase-anon-key="your-anon-key"
  initial-view="sign_in_with_password"
  locale="es"
  auth-options='{"auth": {"email": true, "phone": false}}'
  texts='{"signIn": "Iniciar sesión", "signUp": "Registrarse"}'
></supabase-auth>

Accessing User State

Access the current user state via JavaScript:

const authElement = document.querySelector('supabase-auth');
console.log('Current user:', authElement.user);

// Listen for auth state changes
authElement.addEventListener('auth-state-change', (event) => {
  console.log('Auth state changed:', event.detail);
});
