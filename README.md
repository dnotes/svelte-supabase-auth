# Svelte Supabase Auth

This is the monorepo for development of svelte-supabase-auth, an out-of-the-box single import solution for auth and user management of Supabase projects, built in Svelte and available as a Svelte component or a web component custom element `<supabase-auth>`.

Please see the [main package] for implementation details, or visit the [demo site] to try out the functionality.

[main package]: https://github.com/dnotes/svelte-supabase-auth/tree/main/packages/main
[demo site]: https://svelte-supabase-auth.pages.dev
